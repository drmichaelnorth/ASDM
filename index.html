Introduction

The AnticipatoRy System Dynamics Networks (ASDM) library is a genetic algorithm system for automatically generating system dynamics models that differ in both their embedded parameters and their fundamental equations while maintaining dimensional consistency. It is intended to be used for anticipatory modeling of systems such as complex networks. A reference for the ASDM API can be found here.

Authors

ARCANE was developed by Michael J. North, Pam Sydelko, and Ignacio Martinez-Moyano at Argonne National Laboratory.
